# Erney's Translate Tool

**Версия:** 1.0.0  
**Платформа:** Windows 10/11 (x64)  
**Технологии:** C# .NET 8, WPF, MVVM

---

## 📖 Описание

**Erney's Translate Tool** — это мощный и удобный инструмент для мгновенного перевода текста с любого участка экрана. Программа использует технологию оптического распознавания символов (OCR) для извлечения текста из изображений и автоматически переводит его на выбранный язык.

### Основные возможности

- 🖼️ **Захват области экрана** — выделите любую область для распознавания текста
- 🔤 **Распознавание текста (OCR)** — поддержка Tesseract OCR и Windows.Media.OCR
- 🌐 **Многоязычный перевод** — Google Translate, DeepL, Yandex Translate
- 🔲 **Оверлей** — отображение перевода поверх других приложений
- ⌨️ **Горячие клавиши** — быстрый доступ ко всем функциям
- 📜 **История переводов** — поиск, фильтрация, экспорт
- 💾 **Кэширование** — ускорение работы за счёт сохранения предыдущих переводов
- 🎨 **Тёмная тема** — современный дизайн с настраиваемым интерфейсом

## 📥 Установка

### Вариант 1: Установка через инсталлятор (рекомендуется)

1. Скачайте установочный файл `ErneyTranslateTool-Setup-1.0.0.exe`
2. Запустите установщик от имени администратора
3. Следуйте инструкциям мастера установки
4. При необходимости инсталлятор автоматически установит .NET 8.0 Runtime
5. После завершения установки программа запустится автоматически

### Вариант 2: Портативная версия

1. Скачайте архив `ErneyTranslateTool-1.0.0-Portable.zip`
2. Распакуйте архив в любую папку
3. Запустите `ErneyTranslateTool.exe`
4. При первом запуске убедитесь, что установлен .NET 8.0 Runtime

### Установка .NET 8.0 Runtime

Если .NET 8.0 Runtime не установлен, скачайте его с официального сайта:
- [Скачать .NET 8.0 Runtime](https://dotnet.microsoft.com/download/dotnet/8.0/runtime)
- Выберите версию для Windows x64
- Запустите установщик и следуйте инструкциям

## 🔑 Настройка API ключей

Для работы переводчиков необходимо настроить API ключи.

### Google Translate API

1. Перейдите в [Google Cloud Console](https://console.cloud.google.com/)
2. Создайте новый проект или выберите существующий
3. Включите API Cloud Translation
4. Создайте учётные данные (API ключ)
5. Скопируйте ключ и вставьте в настройки программы

**Бесплатный лимит:** 500 000 символов в месяц

### DeepL API

1. Зарегистрируйтесь на [DeepL](https://www.deepl.com/)
2. Перейдите в раздел [API Keys](https://www.deepl.com/account/summary)
3. Создайте новый API ключ
4. Скопируйте ключ и вставьте в настройки программы

**Бесплатный лимит:** 500 000 символов в месяц

### Yandex Translate API

1. Перейдите в [Yandex Cloud](https://cloud.yandex.ru/)
2. Создайте новый каталог или выберите существующий
3. Включите API Translate
4. Создайте сервисный аккаунт и получите API ключ
5. Скопируйте ключ и вставьте в настройки программы

**Бесплатный лимит:** 1 000 000 символов в месяц

## ⌨️ Горячие клавиши

| Действие | Комбинация | Описание |
|----------|------------|----------|
| **Захват области** | `Ctrl + Alt + Z` | Выделение области экрана для распознавания |
| **Перевод** | `Ctrl + Alt + X` | Перевод распознанного текста |
| **Показать/скрыть оверлей** | `Ctrl + Alt + O` | Отображение окна с переводом |
| **Скрыть оверлей** | `Escape` | Быстрое скрытие оверлея |

> ⚠️ Горячие клавиши можно изменить в настройках программы.

## 🎮 Использование в играх

### Инструкция

1. Запустите игру в оконном или полноэкранном режиме
2. Нажмите `Ctrl + Alt + Z` для захвата области с текстом
3. Выделите область, содержащую текст для перевода
4. Нажмите `Ctrl + Alt + X` для перевода
5. Перевод отобразится в оверлее поверх игры

### Настройка оверлея для игр

1. Откройте настройки → вкладка "Оверлей"
2. Установите позицию "Нижний центр" или удобную вам
3. Включите опцию "Всегда поверх других окон"
4. Включите опцию "Пропускать клики" для взаимодействия с игрой
5. Настройте прозрачность для лучшей видимости

## 🔧 Настройка и конфигурация

### Файл конфигурации

Настройки хранятся в файле `appsettings.json` в папке пользователя:
```
%APPDATA%\ErneyTranslateTool\appsettings.json
```

### Основные параметры

```json
{
  "TranslationProvider": "Google Translate",
  "GoogleApiKey": "",
  "DeepLApiKey": "",
  "YandexApiKey": "",
  "SourceLanguageCode": "auto",
  "TargetLanguageCode": "ru",
  "CaptureDelay": 500,
  "OcrEngine": "Tesseract",
  "OverlayOpacity": 0.8,
  "OverlayFontSize": 14,
  "OverlayPosition": "BottomRight",
  "AutoStart": false,
  "MinimizeToTray": true,
  "CacheSize": 1000,
  "HistoryLimit": 100
}
```

## 🐛 Решение проблем

### Программа не запускается

1. Убедитесь, что установлен .NET 8.0 Runtime
2. Запустите от имени администратора
3. Проверьте антивирус (может блокировать)
4. Переустановите программу

### Не распознаётся текст

1. Увеличьте масштабирование OCR в настройках (2-3x)
2. Включите предварительную обработку изображения
3. Убедитесь, что текст чёткий и контрастный
4. Попробуйте другой движок OCR

### Не работает перевод

1. Проверьте API ключ в настройках
2. Убедитесь в наличии интернета
3. Проверьте лимиты API сервиса
4. Попробуйте другой сервис перевода

### Не работают горячие клавиши

1. Проверьте, не заняты ли клавиши другими программами
2. Перезапустите программу
3. Измените комбинацию в настройках
4. Запустите от имени администратора

### Оверлей не отображается

1. Проверьте настройку прозрачности
2. Убедитесь, что включена опция "Всегда поверх окон"
3. Перезапустите программу
4. Обновите драйверы видеокарты
