# Erney's Translate Tool

**Версия:** 1.0.0  
**Платформа:** Windows 10/11 (x64)  
**Технологии:** C# .NET 8, WPF, MVVM

---

## 📖 Описание

**Erney's Translate Tool** — инструмент для перевода текста прямо с экрана в режиме реального времени. Программа захватывает указанное окно игры, распознаёт текст с помощью встроенного Windows OCR и отображает перевод поверх оригинала в виде полупрозрачного оверлея. Разработана специально для игр, которые не переведены на русский язык.

### Основные возможности

- 🎮 **Захват окна игры** — привязка к конкретному окну, не мешает другим приложениям
- 🔤 **OCR на базе Windows** — встроенный Windows.Media.OCR, не требует сторонних движков
- 🌐 **Перевод через DeepL API** — высокое качество перевода, автоопределение языка
- 🔲 **Умный оверлей** — перевод накладывается поверх оригинального текста с затемнением
- ♻️ **Кэш переводов** — повторяющийся текст не тратит API-лимит
- ⌨️ **Глобальные горячие клавиши** — управление без переключения из игры
- 📜 **История переводов** — по сессиям с именем игры, поиск и экспорт
- 🎨 **Тёмная тема** — современный интерфейс полностью на русском языке

## 🔑 Настройка DeepL API

Программа работает **только с DeepL API**. Ключ бесплатный — карта не нужна.

### Как получить бесплатный ключ

1. Перейдите на [https://www.deepl.com/pro-api](https://www.deepl.com/pro-api)
2. Нажмите **"Sign up for free"**
3. Зарегистрируйтесь (достаточно email)
4. В личном кабинете перейдите в раздел **Account → API Keys**
5. Скопируйте ключ (заканчивается на `:fx` для бесплатного тарифа)

**Бесплатный лимит:** 500 000 символов в месяц — для личного использования в играх этого более чем достаточно.

### Как ввести ключ в программе

1. Откройте программу
2. Перейдите на вкладку **«Настройки перевода»**
3. Вставьте ключ в поле **«API-ключ DeepL»**
4. Нажмите **«Проверить ключ»** — программа подтвердит корректность
5. Ключ сохраняется зашифрованно (Windows DPAPI)

## 🎮 Использование в играх

1. Запустите игру в **оконном** или **безрамочном (borderless)** режиме
2. Запустите **Erney's Translate Tool**
3. На вкладке **«Главная»** нажмите **«Выбрать окно»** и выберите окно игры
4. Нажмите **«Запустить»** или используйте `Ctrl + Shift + T`
5. Программа автоматически найдёт текст на экране и наложит перевод поверх него
6. При переключении из игры оверлей автоматически прекращает работу — переводится только окно игры

### Советы

- Используйте **безрамочный полноэкранный режим** — он работает лучше всего
- Если текст мелкий — увеличьте масштаб интерфейса игры
- Для японского/китайского текста потребуется установить языковые пакеты Windows (см. ниже)

---

## 🌍 Языковые пакеты для OCR

Windows.Media.OCR поддерживает японский и китайский только при наличии соответствующих языковых пакетов системы. Английский работает из коробки.

### Установка языковых пакетов

1. Откройте **Параметры Windows** → **Время и язык** → **Язык и регион**
2. Нажмите **«Добавить язык»**
3. Найдите и добавьте нужный язык:
   - **日本語** — для японского
   - **中文 (简体)** — для упрощённого китайского
   - **中文 (繁體)** — для традиционного китайского
4. При установке убедитесь, что отмечен пункт **«Пакет распознавания рукописного ввода»** (он включает OCR)
5. После установки **перезапустите** программу

Программа автоматически определит отсутствие нужных пакетов и покажет предупреждение с инструкцией.

## 🖥️ Описание вкладок

### Главная
Основное управление: выбор окна игры, запуск/остановка перевода, статус работы и статистика (переведено символов, процент попаданий в кэш).

### Настройки перевода
Ввод и проверка API-ключа DeepL, выбор целевого языка перевода, информация об установленных языковых пакетах OCR.

### Настройки оверлея
Выбор шрифта из системных, режим размера (авто/вручную), прозрачность фона, цвета текста и фона, предпросмотр внешнего вида.

### История
История переводов по игровым сессиям (название берётся из заголовка окна игры). Поиск, фильтрация, экспорт в `.txt` или `.csv`, очистка истории.

### О программе
Версия, ссылки, кнопки открытия лога и очистки кэша, информация о совместимости с античитами.

## ⚠️ Совместимость с античитами

Программа использует **стандартный Windows API захвата экрана (Windows Graphics Capture)**. Она **не изменяет память игры** и **не внедряет код** в игровые процессы.

**Для одиночных игр:** полностью безопасно.

**Для онлайн-игр с мультиплеером:** использование может нарушать условия использования игры. Используйте на свой страх и риск.

### Что программа НЕ делает:
- ❌ Не использует драйверы уровня ядра
- ❌ Не внедряется в процессы игр
- ❌ Не перехватывает вызовы DirectX/OpenGL
- ❌ Не изменяет игровую память

### Игры с повышенным риском (онлайн с античитом):
- Valorant (Vanguard), Call of Duty (Ricochet), Fortnite (EAC), Apex Legends (EAC), PUBG (BattlEye)

### Безопасное использование:
- ✅ Одиночные игры без онлайн-компонентов
- ✅ Визуальные новеллы
- ✅ Старые игры без античита

## 🐛 Решение проблем

### Программа не запускается
- Убедитесь, что установлен .NET 8.0 Runtime
- Запустите от имени администратора

### Текст не распознаётся
- Убедитесь, что игра запущена в оконном/безрамочном режиме
- Для японского/китайского установите языковые пакеты Windows (см. выше)
- Текст должен быть чётким, не слишком мелким

### Оверлей не отображается
- Убедитесь, что перевод запущен (`Ctrl + Shift + T`)
- Проверьте, не скрыт ли оверлей (`Ctrl + Shift + H`)
- Убедитесь, что выбрано верное окно игры

### Не работает перевод
- Нажмите **«Проверить ключ»** на вкладке «Настройки перевода»
- Убедитесь в наличии интернета
- Проверьте остаток лимита на [deepl.com](https://www.deepl.com/account/summary)

### Не работают горячие клавиши
- Проверьте, не заняты ли клавиши другими программами
- Перезапустите программу от имени администратора
